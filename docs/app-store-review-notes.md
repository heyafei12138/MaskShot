# App Store Review Notes

MaskShot v1.0 is a free, local-first screenshot privacy tool.

- No Paywall, StoreKit, IAP, Restore Purchase, or Unlock Pro flow is present in v1.0.
- The app does not create accounts.
- The app does not upload user images.
- Photo permission is requested only for finding the latest screenshot.
- Manual image selection uses the system picker.
- Demo screenshots should use fictional sample content only.
- Privacy Policy: `docs/privacy-policy.md`
- Terms of Use: `docs/terms-of-use.md`
